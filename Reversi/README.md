## 開発プログラム
リバーシのプログラムです。<br>
黒石をプレイヤー・白石がAIです。<br>
AIはMiniMaxの探索アルゴリズムで打つ手を決めます。<br>

## 開発環境
OS:            Windows10<br>
Visual Studio: Visual Studio Community 2017<br>

## 動作環境
コマンドラインで実行可能です。<br>

## 注意点
キーボードで入力出来るキーを限定しています。<br>
・入力出来るキー一覧<br>
　①数字<br>
　②アルファベットのa/b/c/d/e/f/g/h/j/k/r/u<br>
　③Enter<br>
　④BackSpace<br>
